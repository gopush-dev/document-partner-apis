2. API 호출 방식

API 연동방식

로그인 연동 API 를 직접 호출하는 방식입니다.

연동 단계

0. 연동 정보 요청

연동에 앞서 아래 정보를 관리자에게 요청해주세요.

  • API_KEY: Gopush API 키
  • key: AES256 알고리즘에 사용할 대칭키

1. 로그인 후 앱에 디바이스 정보 요청

자바스크립트 인터페이스를 통해 현재 로그인한 디바이스 정보를 요청합니다.

const os = android.getDeviceOS()
const platformID = android.getDevicePlatformID()
const deviceID = android.getDeviceUUID()

ReturnType

name type required 설명
uuid string v 기기의 UUID
platform_unique_id string v 앱 platform_unique_id
os ios or android v OS

2. uuid, platform_id 복호화

*복호화키와 결과물 데이터는 클라이언트 사이드에 노출되면 안됩니다.

API 서버에서 복호화 // Good

서버사이드 렌더링 // Good

Javascript // Bad

const uuid = decryptAES256(uuid, key)
const platformUniqueId = decryptAES256(platform_unique_id, key)

3. 로그인 API 호출

  • API_KEY: 발급받은 API Key
  • PLATFORM_UNIQUE_ID: 이전 단계에서 복호화한 platform_id
  • UUID: 이전 단계에서 복호화한 UUID
  • USER_ID: 로그인한 유저 ID
{    
    "id":1,     
    "jsonrpc":"2.0",
    "method":"Device.Update",     
    "params":{
        "api_key": "{API_KEY}",
        "platform_unique_id": "{PLATFORM_UNIQUE_ID}",
        "payload": {
            "uuid": "{UUID}",
            "user_id": "{USER_ID}"
        }
    }
}
